HƯỚNG DẪN GỘP CODE
1. Ý nghĩa các Folder:

	- Patch: phần được sửa đổi
2. Gộp file code:

	- Vào đường dẫn Patch/code: chạy file merge.bat

	- Vào Patch/dataBase/ có file run.sql. Import nó vào database chính dự án
	- Lưu ý: 2 bảng được sửa đổi : comment, tbl_user
